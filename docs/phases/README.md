## How To Use These Briefs

1. read the master plan first,
2. read [PHASE_GOVERNANCE.md](C:/Users/Administrator/OneDrive/Documents/co_ma/docs/phases/PHASE_GOVERNANCE.md) before closing or opening any phase,
3. use the phase brief for the phase you are currently working in,
4. use the closeout and entry docs when changing phases,
5. use the templates when a new phase-transition document needs to be created,
6. use `TODO.md` as the active checklist,
7. do not treat later-phase briefs as permission to jump ahead.
